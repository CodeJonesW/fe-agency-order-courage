# fe-game-1

Frontend for Game 1 - a minimal RPG-style quest interface.

## Setup

```bash
npm install
```

## Development

Start the frontend dev server:

```bash
npm run dev
```

The frontend will run on `http://localhost:5173` and proxy `/api/*` requests to the backend at `http://localhost:8787`.

**Important:** Make sure the backend is running first:

```bash
cd ../be-game-1
npm run dev
```

## Build

```bash
npm run build
```

## Preview

```bash
npm run preview
```
